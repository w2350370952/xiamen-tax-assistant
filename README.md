# 厦国会税务专硕助手 · EdgeOne 镜像

面向中国大陆访问的课程管理镜像。学生无需登录即可查看课表；管理员可在镜像站内完成账号密码登录、PDF解析、待审核课程校对、发布和文件删除。课程数据优先从正式课程系统实时同步，连接异常时自动使用内置课程备份。

- 构建命令：`npm run build`
- 输出目录：`dist`
- 管理后台通过 EdgeOne 同域函数安全连接课程服务，浏览器无需访问原站域名

## 财经模块 2.0（纳斯达克100投资分析助手）

- 金融仪表盘：实时点位、今日涨跌、近1月/3月/1年表现、市场状态、风险星级、投资建议
- 历史走势：1个月至10年七档周期，折线/K线切换，纳斯达克100与上证指数双Y轴真实点位对比
- 估值分析：实时PE（Twelve Data）→ 管理员手动录入 → 模型估算三级兜底，页面明确标注数据来源
- 估值刻度条与投资策略区间均由管理员在后台「财经设置」中维护，无需改代码
- 数据来自 Yahoo Finance / Twelve Data，Pages Blob 缓存10年日线，行情接口自动增量更新
- 管理接口：`GET/PUT /api/admin/finance-settings`（需管理员登录）
